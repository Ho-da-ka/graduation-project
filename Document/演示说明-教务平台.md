# ZF青少年体能培训教务管理平台-演示说明（现场版）

文档用途：用于毕业设计现场演示，覆盖后端服务、管理后台、微信小程序三端联动。

## 1. 演示目标
- 展示系统可正常登录并基于角色访问。
- 展示管理后台核心业务：学员、课程、教练、考勤、体测、训练记录。
- 展示小程序端多角色入口（管理员/教练/学生/家长）与退出登录。
- 展示 JWT 鉴权链路（登录、鉴权访问、退出）。

## 2. 环境准备
### 2.1 必备软件
- JDK 17
- MySQL 8.x（数据库：`management_platform`）
- Node.js 18+
- 微信开发者工具

### 2.2 项目目录
- 后端：`D:\GraduationProject\Code\backend-service\ManagementPlatform`
- 管理后台：`D:\GraduationProject\Code\admin-panel`
- 小程序：`D:\GraduationProject\Code\mobile-app`

## 3. 启动步骤（建议顺序）
### 3.1 启动后端（端口 8080）
在 PowerShell 执行：

```powershell
cd D:\GraduationProject\Code\backend-service\ManagementPlatform
$env:JAVA_HOME='C:\Program Files\Java\jdk-17'
$env:MAVEN_USER_HOME='D:\m2repo'
.\mvnw.cmd spring-boot:run
```

验证健康接口：

```powershell
Invoke-WebRequest -UseBasicParsing http://localhost:8080/api/v1/public/ping
```

返回 `success=true` 且 `status=UP` 即正常。

### 3.2 启动管理后台（端口 5173）
另开一个 PowerShell：

```powershell
cd D:\GraduationProject\Code\admin-panel
npm install
npm run dev -- --host 0.0.0.0 --port 5173
```

浏览器访问：`http://localhost:5173`

### 3.3 构建并启动小程序
在 PowerShell 执行：

```powershell
cd D:\GraduationProject\Code\mobile-app
npm install
npm run build:mp-weixin
```

微信开发者工具导入目录：
- `D:\GraduationProject\Code\mobile-app\dist\build\mp-weixin`

## 5. 现场演示脚本（建议按此顺序）
### 5.1 管理后台演示（浏览器）
1. 登录 `admin` 账号。
2. 进入“学员管理”：演示查询、新增、编辑、详情。
3. 进入“课程管理”：演示查询、新增、编辑。
4. 进入“教练管理”：演示教练 CRUD。
5. 进入“考勤管理”：新增考勤并按条件查询。
6. 进入“体测管理”：按学员查询并新增体测。
7. 进入“训练记录”：查看列表、详情、新增/编辑。
8. 点击退出登录。

### 5.2 小程序演示（微信开发者工具）
1. 打开登录页，使用 `student` 登录。
2. 展示学生首页：我的课程、训练记录、体测记录。
3. 点击“退出登录”，返回登录页。
4. 使用 `parent` 登录。
5. 展示家长首页：我的孩子、可约课程、预约记录、签到记录、体测结果、站内消息。
6. 点击“退出登录”，返回登录页。

### 5.3 JWT 鉴权说明（口述建议）
- 系统已从 Basic 迁移为 JWT。
- 登录接口：`POST /api/v1/auth/login`
- 刷新接口：`POST /api/v1/auth/refresh`
- 退出接口：`POST /api/v1/auth/logout`
- 受保护接口通过 `Authorization: Bearer <token>` 访问。

## 6. 常见问题与快速处理
### 6.1 小程序启动后直接进学生页
原因：本地仍有历史登录缓存。

处理：
1. 微信开发者工具 Console 执行：`wx.clearStorageSync()`
2. 重新编译。

### 6.2 小程序报错 `webapi_getwxaasyncsecinfo:fail`
该报错通常为微信开发者工具环境问题（非业务代码逻辑错误）。

处理：
1. 开发者工具重新登录账号。
2. 清缓存（清除全部缓存）并重启工具。
3. 确认 `manifest.json` 中使用有效 `mp-weixin.appid`。

### 6.3 前端请求 401/403
- 401：未登录或 token 过期，重新登录。
- 403：角色无权限，切换到有权限账号演示。

### 6.4 后端启动失败（端口占用）
检查并结束占用进程：

```powershell
netstat -ano | findstr :8080
Stop-Process -Id <PID> -Force
```

## 7. 演示结束后关闭服务
```powershell
# 后端
Stop-Process -Id <backend_pid> -Force

# 管理后台前端
Stop-Process -Id <frontend_pid> -Force
```
