# 设计规格：管理端极简工作台美化 (方案 A)

## 1. 目标与定位
*   **目标读者**：机构管理员、教练
*   **视觉风格**：极简工作台 (Minimalist Workspace)
*   **核心理念**：数据驱动决策。将杂乱的管理入口转化为有序的数据仪表盘，提升信息获取效率。

## 2. 视觉规范 (UI Guidelines)

### 2.1 色彩方案
| 用途 | 颜色值 | 说明 |
| :--- | :--- | :--- |
| 背景 | `#F8FAFC` | 极浅灰背景，增加页面深度 |
| 内容背景 | `#FFFFFF` | 纯白卡片 |
| 主品牌色 | `#3B82F6` | 睿智蓝，代表专业与技术 |
| 标题文字 | `#0F172A` | 深石板色，沉稳可靠 |
| 警告/重要 | `#EF4444` | 醒目的错误/退出色 |

### 2.2 核心布局：数据仪表盘 (Dashboard)

#### 首页 (Home)
*   **状态概览 (Top Statistics)**：
    *   将原本的 2x2 格子改为更具层次感的布局。
    *   **主磁贴**：学员总数、课程总数。使用大字号、深蓝文字。
    *   **次磁贴**：考勤、体测、训练记录。使用较小字号，搭配极细边框。
*   **功能导航 (Menu Grid)**：
    *   不再使用全宽卡片，而是采用 **2列网格** 磁贴布局。
    *   每个磁贴包含：一个精细的单色图标、标题和简短的功能描述。
*   **状态监测**：将“后端服务状态”移至页面底部或个人中心，保持首页纯净。

#### 列表页 (List Pages)
*   **顶部筛选区**：收纳进一个轻量级的 `u-search` 组件，右侧带一个极简的“筛选”图标，点击弹出抽屉选择状态。
*   **列表项卡片**：
    *   **姓名/标题**：加粗显示。
    *   **标签化状态**：如“正常”显示为浅蓝背景蓝字，“停课”显示为浅灰背景灰字。
    *   **操作区**：简化为卡片底部的两个微型文字按钮，增加点击热区。

## 3. 实现建议
*   **全局样式**：在 `uni.scss` 中统一定义 `$admin-primary-color: #3B82F6`。
*   **组件重构**：新建 `AdminStatTile` (统计磁贴) 和 `AdminMenuTile` (菜单磁贴)。
*   **体验升级**：引入 `u-skeleton` 骨架屏，在数据刷新时提供更平滑的视觉过渡。
