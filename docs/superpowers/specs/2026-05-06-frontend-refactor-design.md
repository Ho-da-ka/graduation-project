# ZF 青少年体能培训教务管理平台 - 前端重构设计规范 (Modern Professional)

## 1. 概述
本规范旨在将现有的管理后台重构为“现代专业感”视觉风格，同时优化交互流程以提升操作效率。重构将围绕“视觉减法”和“功能加焦”展开。

## 2. 视觉规范 (Visual Identity)

### 2.1 色彩体系
*   **基础背景**: `#F8FAFC` (Slate-50) - 用于全局页面背景，营造通透感。
*   **内容背景**: `#FFFFFF` (White) - 用于卡片、侧边栏和表单容器。
*   **品牌主色**: `#3B82F6` (Blue-500) - 用于主按钮、激活状态、关键链接。
*   **品牌辅助色**: `#8B5CF6` (Violet-500) - 专门用于 AI 相关功能，体现“智慧”属性。
*   **文字颜色**:
    *   主标题: `#0F172A` (Slate-900)
    *   正文/标签: `#475569` (Slate-600)
    *   辅助文字: `#94A3B8` (Slate-400)
*   **功能色**:
    *   成功/提升: `#10B981` (Emerald-500)
    *   警告/风险: `#F43F5E` (Rose-500)

### 2.2 形状与间距
*   **圆角 (Border Radius)**:
    *   标准卡片: `16px`
    *   按钮/输入框: `8px`
    *   小标签 (Tag): `6px`
*   **投影 (Shadow)**:
    *   标准卡片: `0 4px 6px -1px rgba(0, 0, 0, 0.05), 0 2px 4px -1px rgba(0, 0, 0, 0.03)`
    *   激活/悬浮状态: `0 10px 15px -3px rgba(0, 0, 0, 0.08)`
*   **间距 (Spacing)**:
    *   页面边距: `32px`
    *   卡片间距: `24px`

## 3. 核心重构模块设计

### 3.1 全局布局 (Layout)
*   **侧边栏 (Sidebar)**: 切换为纯白背景，激活态采用蓝色卡片式高亮。
*   **顶栏 (Header)**: 增加全局搜索框与通知中心，将“快速操作”按钮组集成在右侧。
*   **导航**: 引入面包屑导航，增强路径感。

### 3.2 仪表盘 (Dashboard)
*   **布局**: 采用 2:1 左右分栏。
*   **左侧 (数据区)**: 展示大尺寸 ECharts 面积图与条形图。
*   **右侧 (互动区)**: 垂直展示“待办事项”与“关怀提醒”Feed 流。
*   **统计卡片**: 去除边框，强化数字字重。

### 3.3 AI 智慧中心 (AI Hub)
*   **风格**: 保持浅色专业版，但增加渐变点缀。
*   **交互**: 强化“生成报告”过程的实时反馈（如加载动效、数据扫描感）。

## 4. 技术实施路径
1.  **全局样式覆写**: 更新 `styles.css`，定义全局 CSS 变量与 Element Plus 组件覆写。
2.  **布局重构**: 修改 `AdminLayout.vue`，实现浅色卡片式侧边栏。
3.  **核心视图重写**: 依次重构 `DashboardView.vue` 和 `AiAnalysisHub.vue`。
4.  **图表定制**: 封装通用的 ECharts 主题配置，移除冗余装饰元素。
