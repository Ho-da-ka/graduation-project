# 设计规格：Web 端视觉重塑与全平台 AI 集成

## 1. 目标与定位
*   **目标**：提升 Web 后台的“工具质感”，并通过 AI 将零散的训练数据转化为有价值的成长洞察。
*   **视觉风格**：Premium Minimalism (高端极简)。
*   **AI 核心**：学员成长分析、自动评语生成、智慧分析中心。

## 2. Web 端 (Admin Panel) 视觉规范

### 2.1 全局视觉升级
*   **背景色**：`#F8FAFC` (微冷灰)，增加页面透气感。
*   **卡片样式**：白底、12px 圆角、`box-shadow: 0 4px 15px rgba(0,0,0,0.02)`。
*   **侧边栏**：深色调 (`#0F172A`) 切换为轻盈的浅色悬浮风格，或极简深蓝色。
*   **字体**：主标题采用 Semibold，正文采用常规字重，强调信息层级。

### 2.2 AI 入口设计 (Entry Points)
1.  **学员详情页 (AI Insights Card)**：
    *   **位置**：档案顶部全宽区域。
    *   **表现**：紫色微渐变边框，展示“AI 智能综述”。
    *   **功能**：展示爆发力、耐力等维度的 AI 评估。
2.  **录入页助手 (AI Magic Pen)**：
    *   **位置**：体测/训练记录的“评语”输入框右侧。
    *   **功能**：点击后根据当前输入的数值，调用后台 AI 接口自动生成一段 100 字左右的专业寄语。
3.  **智慧中心 (AI Hub)**：
    *   **入口**：侧边栏新增“AI 分析”独立频道。
    *   **功能**：按班级或时间段生成深度分析报告。

## 3. 小程序端 (Mini-Program) AI 集成

### 3.1 AI 智能简报 (Dashboard Feed)
*   **表现形式**：在家长端首页“时间轴”或“成长总览”顶部插入 AI 简报卡片。
*   **视觉细节**：
    *   左上角带 `AI 分析` 专属芯片 (Chip) 标签。
    *   内容采用“流式打字机”动画效果加载，增加仪式感。
*   **交互**：点击卡片可查看完整的“本月成长深度解析”。

## 4. 技术方案建议
*   **前端**：
    *   Web 端：使用 Element Plus 的 `v-loading` 配合 AI 生成状态。
    *   小程序：使用 `up-transition` 实现卡片的优雅进入。
*   **后端联动**：对接 `GeneratedContentService`，利用其 AI 生成逻辑。

## 5. 实施计划 (Roadmap)
*   **Phase 1**: Web 端全局视觉重塑 (CSS/Layout 优化)。
*   **Phase 2**: 开发 Web 端 AI 入口组件及交互。
*   **Phase 3**: 实现小程序端 AI 简报展示逻辑。
