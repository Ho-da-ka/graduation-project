# 设计规格：家长首页极简专业化优化 (方案 A)

## 1. 目标与定位
*   **目标读者**：培训机构家长
*   **视觉风格**：极简专业 (Minimalist Professional)
*   **核心理念**：效率至上，消除冗余，通过时间轴构建“日程管理”+“价值交付”的闭环。

## 2. 视觉规范 (UI Guidelines)

### 2.1 色彩方案
| 用途 | 颜色值 | 说明 |
| :--- | :--- | :--- |
| 背景 | `#FFFFFF` | 纯白背景，增加空间感 |
| 标题文字 | `#0F172A` | 深石板色，提供最高对比度 |
| 正文/描述 | `#475569` | 冷灰色，用于次要信息 |
| 品牌主色 | `#2563EB` | 品牌蓝，用于高亮和操作 |
| 辅助背景 | `#F8FAFC` | 极浅灰，用于已完成日程的轻微遮罩 |
| 分隔线 | `#E2E8F0` | 用于时间轴主干 |

### 2.2 字体排版
*   **字号层级**：
    *   课程标题：34rpx, Weight 600
    *   时间/地点：26rpx, Weight 400
    *   反馈链接：28rpx, Weight 500 (蓝色)
*   **间距**：页面边距保持 32rpx，卡片内间距 24rpx。

## 3. 核心布局：极简日程流 (Pure Timeline)

### 3.1 吸顶日期条 (Sticky DateStrip)
*   **行为**：横向滚动，自动定位到今天。
*   **视觉**：无背景色，仅数字。选中项数字变大并带有一条 4rpx 厚度的短横线。

### 3.2 垂直时间轴 (Vertical Timeline)
*   **左侧 (Time Column)**：
    *   显示具体时间点（如 `14:30`）。
    *   垂直线贯穿所有日程。
*   **中间 (Node)**：
    *   **待开始**：8rpx 实心蓝圆点。
    *   **已完成**：8rpx 空心灰圆圈。
*   **右侧 (Content Card)**：
    *   **待开始卡片**：纯白背景，黑色标题，下方显示教练与教室。
    *   **已完成卡片**：背景切换为 `#F8FAFC`，标题略微置灰。
    *   **核心出口**：在卡片底部右对齐显示蓝色链接 `查看课后反馈报告 →`。

## 4. 交互逻辑
*   **空状态**：若当日无课，显示一行极其淡化的文字：“今日暂无日程安排”，并提供“查看全月课表”按钮。
*   **点击行为**：
    *   点击卡片主体：进入课程详情。
    *   点击“查看课后反馈报告”：直达体能分析/教练评语页面。

## 5. 实现建议 (Tech Strategy)
*   **组件化**：重构 `ParentHomeHero` 为 `ParentDateStrip`，重构 `ParentHomeActivityCard` 为 `TimelineItem`。
*   **性能**：使用 `computed` 处理课程数据的排序与分类，确保时间轴渲染逻辑清晰。
