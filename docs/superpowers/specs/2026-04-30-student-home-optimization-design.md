# 设计规格：学生端首页极简活力优化 (方案 A)

## 1. 目标与定位
*   **目标读者**：体能培训学员
*   **视觉风格**：极简极速 (Minimalist & Direct)
*   **核心理念**：行动导向。通过显著的行动卡片和进度可视化，降低操作成本，激发锻炼动力。

## 2. 视觉规范 (UI Guidelines)

### 2.1 色彩方案
| 用途 | 颜色值 | 说明 |
| :--- | :--- | :--- |
| 背景 | `#FFFFFF` | 纯白背景 |
| 标题文字 | `#0F172A` | 深石板色 |
| 活力主色 | `#F97316` | 活力橙，用于学生端核心按钮和进度 |
| 成功状态 | `#22C55E` | 清爽绿，用于已完成状态 |
| 辅助底色 | `#F8FAFC` | 极浅灰，用于背景区分 |

### 2.2 核心组件：行动中心 (Action Hub)
*   **进度环 (Progress Circle)**：
    *   位于卡片左侧，采用 2px 极细圆环。
    *   橙色展示已完成进度，浅灰色展示剩余。
    *   环中心显示百分比数字。
*   **今日课程 (Today's Course)**：
    *   位于卡片右侧，大号粗体标题。
    *   下方显示时间（如 `18:30`）和场地。
*   **核心按钮 (CTA Button)**：
    *   卡片底部或右下角的圆角按钮。
    *   使用 `linear-gradient(to right, #F97316, #FB923C)`。

## 3. 功能布局
*   **顶部 (Hero Section)**：行动中心卡片。
*   **中部 (Feature Grid)**：
    *   2x2 或 1x3 的极简磁贴。
    *   包含：`我的课程`、`体测报告`、`训练历史`。
    *   图标采用单色细线条风格。
*   **底部 (Footer)**：极简的刷新与个人设置入口。

## 4. 交互逻辑
*   **动态展示**：若今日无课，进度环依然保留，但课程区显示“今日休息，保持活力”，并推荐查看“自主练习计划”。
*   **一键触达**：点击核心按钮直接跳转到当前课程的签到或打卡页面。

## 5. 实现建议
*   **数据驱动**：需要根据学生的所有课程计算本月/本周完成度百分比。
*   **组件化**：新建 `StudentActionHub` 和 `StudentFeatureTile` 组件。
